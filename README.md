# Algorithm-visualization-SDL
Visualization of algorithms in SDL

## Libraries needed
- SDL2

#
Running

`
gcc algorithms.c -O3 -o sort `sdl2-config --libs` -std=c99 -pedantic -ansi && ./sort
`
